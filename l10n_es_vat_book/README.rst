Instalación
===========

Para instalar este modulo necesitas:

* account
* base_vat
* l10n_es
* l10n_es_aeat
* account_refund_original
* account_chart_update
* account_invoice_currency

Se instalan automáticamente si están disponibles en la lista de addons.
